\# Simple Task List – Python Project



\## Description

This project is a simple task management application developed using Python.

It allows users to add, remove, list, prioritize tasks, and receive task recommendations

based on task priority.



\## Features

\- Add tasks

\- Remove tasks

\- List all tasks

\- Priority-based task recommendation

\- File storage using text file



\## Technologies Used

\- Python

\- Lists \& Dictionaries

\- File I/O



\## How to Run

1\. Open terminal

2\. Navigate to project folder

3\. Run:

&#x20;  python task\_manager.py
